# Plan de Campañas Publicitarias - Dulces Creaciones

## Resumen Ejecutivo
Plan de ads para pastelería artesanal en Temperley, Zona Sur GBA.
- **Presupuesto**: $15,000-50,000 ARS/mes
- **Objetivo**: Conseguir más clientes/pedidos
- **Canal de conversión**: WhatsApp

---

## 1. Facebook/Instagram Ads

### Estructura de Campañas

| Campaña | Objetivo | Presupuesto | Duración |
|---------|----------|-------------|----------|
| Campaña 1: Conversiones | Conversiones | $25-35k | Días seleccionados |
| Campaña 2: Tráfico | Tráfico | $10-15k | 2 semanas/mes |
| Campaña 3: Remarketing | Conversiones | $5-10k | Continua |

### Objetivos de Campaña

1. **Conversiones** (PRIMARY) - Generar WhatsApps directos
2. **Tráfico** - Llevar a perfil/sitio, crear audiencias personalizadas
3. **Alcance** - Awareness en audiencias frías

---

### Audiencias Recomendadas

#### Custom Audiences
- **Web Traffic**: Visitantes del sitio (30 días)
- **Seguidores IG**: Personas que siguen el perfil
- **Engagement**: Interacciones con contenido (180 días)
- **Clientes**: Lista de clientes anteriores

#### Lookalike Audiences
1. Lookalike de Clientes (1%) - Más similar a compradores
2. Lookalike de Engaged Users (1%)
3. Basado en audiencia de al menos 100 personas

#### Interest Audiences
- "Torta de cumpleaños", "Pasteles personalizados"
- "Repostería artesanal", "Cupcakes", "Mesa de dulces"
- "Fiesta de niños", "Cumpleaños infantil"
- "Maternidad", "Parenting", "Familia"
- "San Lorenzo", "River", "Boca" (para tortas fútbol)
- "Disney", "Marvel", "Videojuegos", "Anime" (para temáticas)

#### Configuración Demográfica
- **Género**: Femenino (prioritario) + Masculino
- **Edad**: 25-54 años
- **Ubicación**: Temperley, Lomas de Zamora, Banfield, Lanús, Adrogué (radio 20km)

---

### Formatos de Creatives

#### Imagen (Single Image)
- Fotos de tortas terminadas (close-up de decoración)
- Antes/después
- Especificaciones: 1080x1080 (Feed), 1080x1350 (Stories)
- Texto overlay: máximo 20% de la imagen

#### Video (15-30 seg)
- Proceso de decoración (time-lapse)
- Unboxing de torta terminada
- Compilaciones de diseños
- Subtítulos incluidos (85% se ve sin sonido)

#### Stories Ads
- 1080x1920 pixels
- Duración: 7-10 segundos
- Stickers interactivos (encuestas, preguntas)

## Configuración Pixel

### Instalación
1. Crear Facebook Business
2. Obtener código del Pixel desde Events Manager
3. Insertar en todas las páginas del sitio

### events.js para WhatsApp Track
```javascript
function trackWhatsAppClick(ctaLocation) {
  if (typeof fbq !== 'undefined') {
    fbq('track', 'Lead', {
      content_category: 'tortas',
      location: ctaLocation
    });
  }
}
```
