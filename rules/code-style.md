# Reglas de Estilo — Dulces Creaciones

## HTML
- lang="es-AR" en todas las páginas
- Meta tags GEO obligatorios: ai-purpose, ai-summary, ai-faq, llms-txt
- Schema JSON-LD siempre: BreadcrumbList + Service + FAQPage mínimo
- WhatsApp con UTM params siempre
- Canonical URL siempre

## CSS (styles.css)
- Usar variables CSS existentes: --rosa, --rosa-d, --choco, --crema, --nude, --muted
- No crear nuevas variables de color sin necesidad
- Mobile-first

## JS (scripts.js)
- No modificar scripts.js salvo bugs
- WhatsApp tracking via GA4
