# Workflow: Crear Página PSEO

## Cuándo
El usuario pide una página para una localidad, temática o pregunta nueva.

## Pasos
1. Elegir patrón: tortas-adrogue.html (local) o tortas-infantiles.html (temático)
2. Copiar estructura completa
3. Adaptar: title, meta description, h1, keywords, schema, UTM params
4. Agregar hidden link en index.html
5. Agregar al sitemap.xml
6. Validar con geo_audit.py si es aplicable
